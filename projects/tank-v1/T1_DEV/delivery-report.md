# T1_DEV 交付报告 — 坦克大战游戏 MVP

## 概述

完成了坦克大战网页游戏的 MVP 版本，使用纯 HTML5 Canvas + JavaScript（零依赖）实现。

## 交付文件清单

| 文件 | 说明 |
|------|------|
| `index.html` | 入口页面，包含 Canvas 画布、HUD 和 Overlay 元素 |
| `css/style.css` | 暗色主题样式，带发光边框和半透明遮罩 |
| `js/main.js` | 启动入口，初始化输入系统，驱动游戏循环 |
| `js/input.js` | 键盘输入管理器，支持持续按键和单次按键检测 |
| `js/map.js` | 20×15 地图数据，砖墙/钢墙绘制，墙壁碰撞矩形生成 |
| `js/tank.js` | Tank 类，支持移动、射击、受击、冷却，Canvas 旋转绘制 |
| `js/bullet.js` | Bullet 类，飞行、墙壁碰撞（可破坏砖墙）、边界检测 |
| `js/collision.js` | AABB 矩形碰撞检测：坦克-墙壁、坦克-坦克、子弹-坦克 |
| `js/enemy.js` | EnemyController：5 个敌方坦克 AI，随机移动、周期射击 |
| `js/game.js` | Game 状态机：菜单→游戏中→暂停→胜利/失败，完整碰撞循环 |

## 功能特性

- 玩家坦克（绿色）：方向键移动，空格射击
- 5 个敌方坦克（红色）：随机移动 AI + 自动射击
- 两种墙壁：砖墙（可被子弹摧毁）和钢墙（不可摧毁）
- 3 HP 血条系统
- 完整游戏状态机：菜单 → 游戏 → 暂停 → 胜利/失败
- Canvas 2D 旋转绘制（坦克按方向旋转）
- 射击冷却 ~0.5 秒

## 验证步骤

1. 用浏览器打开 `index.html`
2. 按 Enter 开始游戏
3. 方向键控制移动，空格射击
4. 消灭所有 5 个敌方坦克即为胜利
5. 玩家 HP 归零则游戏失败
6. 按 Esc / P 暂停和继续
